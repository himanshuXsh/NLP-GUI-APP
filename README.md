# NLP GUI Application

This is a Python Tkinter based NLP GUI application.

## Features
- Login & Register system
- Named Entity Recognition
- Language Detection
- Sentiment Analysis
- NLPCloud API integration

## Technologies Used
- Python
- Tkinter
- NLPCloud API

## How to Run
1. Install dependencies
2. Add your API key
3. Run nlp_gui.py
